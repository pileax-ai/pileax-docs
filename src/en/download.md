# Installation Instructions

## <svg-icon name="icon-apple" size="28px" /> macOS
1. Download the `.dmg` file for your chip (Apple Silicon or Intel).
2. Open the `.dmg` and drag **PileaX** into your `Applications` folder.
3. Open PileaX from `Applications`.

</div>


<div data-aos="fade-up" data-aos-delay="800">

## <svg-icon name="icon-microsoft-windows" size="28px" color="#0078d4" /> Windows
1. Download the `.exe` installer.
2. Double-click the downloaded file and follow the installer prompts. **PileaX** will be added to your Start menu.

::: warning `SmartScreen warning`
Windows may show a `Windows protected your PC` dialog.

Click `More info` → `Run anyway`. This happens because the app isn't signed with a paid Microsoft certificate.
:::

</div>

<div data-aos="fade-up">

## <svg-icon name="icon-linux" size="28px" /> Linux
1. Download the `.AppImage` file.
2. Double-click the .AppImage, **PileaX** will run without installation.

</div>
