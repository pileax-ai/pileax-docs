# 安装说明

## <svg-icon name="icon-apple" size="28px" /> macOS
1. 根据你的芯片型号（Apple Silicon 或 Intel）下载 `.dmg` 安装文件；
2. 打开 `.dmg` 文件，将 **PileaX** 拖入 `应用程序` 文件夹；
3. 在 `应用程序` 中打开 PileaX。

</div>


<div data-aos="fade-up" data-aos-delay="800">

## <svg-icon name="icon-microsoft-windows" size="28px" color="#0078d4" /> Windows
1. 下载 `.exe` 安装程序；
2. 双击下载好的文件，按照安装向导提示完成安装。**PileaX** 会自动添加到开始菜单。

::: warning `SmartScreen 警告`
Windows 可能会弹出`Windows 已保护你的电脑`提示框。

点击`更多信息`→`仍要运行`。出现该提示是因为此应用未使用付费的微软数字证书签名。
:::

</div>

<div data-aos="fade-up">

## <svg-icon name="icon-linux" size="28px" /> Linux
1. 下载 `.AppImage` 文件。
2. 双击该 `.AppImage` 文件，**PileaX** 无需安装即可直接运行。

</div>
