# <svg-icon name="microsoft-windows" size="36px" color="#0078d4" /> 在 Windows 上安装

## 系统要求
- 系统版本：Windows 10+
- 内存：4GB
- CPU：2核

## 安装步骤

1. 下载 `.exe` 安装程序；
2. 双击下载好的文件，按照安装向导提示完成安装。**PileaX** 会自动添加到开始菜单。

::: warning `SmartScreen 警告`
Windows 可能会弹出`Windows 已保护你的电脑`提示框。

点击`更多信息`→`仍要运行`。出现该提示是因为此应用未使用付费的微软数字证书签名。
:::
